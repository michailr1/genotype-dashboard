🧬 Приватность

Эта страница раскрывает персональные генотипы. Raw VCF не должен попадать в публичный репозиторий; в GitHub Pages достаточно HTML-сводки, если вы осознанно хотите ее публиковать.

⚖️ Интерпретация

VCF содержит GT без глубины/качества. CYP2D6, HLA и структурные варианты требуют специализированного анализа. Нейро-SNP — ассоциации с низкой/средней доказательностью, не диагноз.
