# Genotype Dashboard

Static GitHub Pages dashboard generated from a Genotek VCF key-SNP extraction.

- Entry page: `index.html`
- Dashboard: `dashboard/index.html`
- Raw VCF is **not** included.
- Educational visualization only; not medical advice.

## Privacy

This repository publishes selected genotype/SNP values as an HTML dashboard. Do not add raw genotype files, VCFs, FASTQ/BAM files, or private medical documents.
